- git commit -am "feature commit"

Implemented changes in the feature branch and created a new commit.

## Merge Operations

- git switch master
- git merge feature

Merged feature branch into master using Fast-Forward Merge.

## History Visualization

- git log --oneline --branches --graph

Checked commit history and branch structure.

## Version Recovery

- git revert HEAD --no-edit

Created a new commit that reverts previous changes without deleting commit history.
